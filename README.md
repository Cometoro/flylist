# Flylist project
최신버전 1.0

노래방에서 일본곡의 빠른 검색과 정보를 제공하기 위해 만든 개인용 웹 아카이브.

##주요 파일구성

- `index.html`: 페이지 구조
- `styles.css`: 화면 스타일
- `data.js`: 곡 데이터
- `app.js`: 검색, 필터, 정렬, 즐겨찾기 

## 이용 및 저작권 안내

Flylist는 일본곡 검색과 기록을 목적으로 제작된 비공식·비상업적 웹 아카이브입니다.
수록된 곡들과 반주곡 번호에 대한 모든 권리는 원작자에게 있습니다. 
Flylist는 음원 또는 제3자 상표에 대한 권리를 주장하지 않습니다.
