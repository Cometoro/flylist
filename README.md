# Flylist

노래방에서 일본곡의 빠른 검색과 정보를 제공하기 위해 만든 웹 아카이브.
더이상 예약시간으로 시간을 낭비 할 수 없다.

## 파일구성

- `index.html`: 페이지 구조
- `styles.css`: 화면 스타일
- `data.js`: 곡 데이터
- `app.js`: 검색, 필터, 정렬, 즐겨찾기 

## 이용 및 저작권 안내

Flylist는 개인적인 노래 검색과 기록을 목적으로 제작된 비공식·비상업적 웹 아카이브입니다.
수록된 곡명, 원제, 아티스트명, 작품명 및 노래방 곡 번호 등의 정보에 관한 권리는 
각 권리자 또는 정보 제공자에게 귀속됩니다. Flylist는 음악, 가사, 음원 또는 제3자 상표에 대한 권리를 주장하지 않습니다.

## 업데이트 목록

TJ 데이터와 비교해서 `신규`, `번호 변경`, `수정`으로 판단된 곡은 앱의 `업데이트` 탭에 모아둘 예정입니다.
곡 데이터에는 필요할 때 `updateType`, `updatedAt`, `updateNote` 값을 추가합니다.
